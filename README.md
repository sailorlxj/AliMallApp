# AliMallApp-支付宝小程序

支付宝小程序完美上线；
这是一款已经正式运营的支付宝商城小程序，也是全网第一款商用级并开源出来的支付宝小程序项目。欢迎关注啊！

已上线版本为简洁版，可无限定制你能想要的功能哦；

### 主要功能介绍
整个项目包括两部分：
* 前端支付宝小程序
* 运营后台
